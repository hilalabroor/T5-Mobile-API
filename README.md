### Deskripsi Singkat "Student Contact App"
Aplikasi Android berbasis Kotlin yang mendemonstrasikan implementasi **REST API** menggunakan library **Retrofit**. 
Aplikasi ini memiliki sistem autentikasi berbasis **Token (Bearer Token)** serta dilengkapi dengan fitur **CRUD (Create, Read, Delete)** untuk mengelola data pasien secara *realtime* dari server ke dalam *RecyclerView*.
