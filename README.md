### Deskripsi Singkat "Student Contact App"
Aplikasi Android berbasis Kotlin yang mendemonstrasikan implementasi **REST API** menggunakan library **Retrofit**. 
Aplikasi ini memiliki sistem autentikasi berbasis **Token (Bearer Token)** untuk mengakses data pasien dari server.
